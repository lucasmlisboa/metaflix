# Project: Netflix clone
<h3> About </h3>
<p>This resource was created to put into practice the knowledge acquired in the Digital Inovation One HTML, CSS and Javascript course of a Frontend Web project.
The objective of the project was to recreate an interface of the Netflix project using simple technologies such as HTML5, CSS3, javascript and ReactJS, in addition to encompassing the concepts of how to structure layouts, CSS3 techniques with containers and variables.</p>
